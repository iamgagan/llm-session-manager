üõ†Ô∏è Manual Installation

  1. Install prerequisites:
    • Python 3.10+
    • Poetry (Python package manager)
    • Node.js 18+ (for collaboration features)
  2. Clone repository:
    git clone https://github.com/yourusername/llm-session-manager.git
    cd llm-session-manager
  3. Install dependencies:
    poetry install
  4. Initialize setup:
    poetry run python -m llm_session_manager.cli init
  5. Start monitoring:
    poetry run python -m llm_session_manager.cli monitor
